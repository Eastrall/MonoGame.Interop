MonoGame.Interop
====

MonoGame.Interop is a small library who enables you to use MonoGame into a WPF window.
Only availiable for DirectX plateforms.

Current status
----

MonoGame.Interop handles:

- Rendering (2D and 3D)
- Mouse input
